# 传奇霸主自动化脚本 - AutoJS版

基于原始油猴脚本 [ChuanQiBazhuScript-main](../ChuanQiBazhuScript-main/) 移植的 AutoJS 版本。

## 功能特性

- **自动战斗** - 自动寻找并攻击怪物，支持AoE拉怪
- **自动地图切换** - 根据配置自动切换到最佳打宝地图
- **反挂机答题** - 自动识别并解答诗词找字、数学运算问题
- **背包管理** - 自动整理、出售、存仓物品
- **日常任务** - 自动领取VIP奖励、购买商店、完成任务
- **拍卖扫拍** - 自动竞拍有价值的物品

## 环境要求

- Android 7.0 或更高版本
- 已安装 AutoJS (Pro版或免费版)
- 微信小程序已打开游戏

## 安装步骤

### 1. 安装 AutoJS

从以下渠道安装 AutoJS:
- AutoJS官方: https://github.com/topjohnwu/AutoJS6
- 或在手机上安装APK

### 2. 导入脚本

方法一: 直接复制
```bash
# 将 auto-agent-bz 文件夹复制到手机存储
adb push auto-agent-bz /sdcard/脚本/auto-agent-bz/
```

方法二: 使用 AutoJS 的项目功能
- 在 AutoJS 中创建新项目
- 将本文件夹内容同步到手机

### 3. 配置

编辑 `main.js` 中的 CONFIG 对象:

```javascript
const CONFIG = {
  mode: 1,           // 0=元宝模式, 1=打玉模式, 2=大号模式
  combat: {
    enabled: true,
    attackRange: 6,
    priorityKeywords: ['至尊', 'BOSS'],
  },
  antiAfk: {
    enabled: true,
  },
  daily: {
    enabled: true,
    vipReward: true,
    shopBuy: true,
  }
};
```

### 4. 运行

1. 打开微信，进入传奇霸主游戏
2. 打开 AutoJS
3. 选择 `main.js`
4. 点击运行按钮

## 使用说明

### 首次使用

1. 运行脚本前确保已在游戏中
2. 首次运行会请求截图权限，请授权
3. 建议先在模拟器上测试

### 模式说明

| 模式 | 名称 | 适用场景 |
|------|------|----------|
| 0 | 元宝模式 | 低级号刷元宝 |
| 1 | 打玉模式 | 中级号打玉 |
| 2 | 大号模式 | 高级号刷反伤BOSS |

### 常用操作

- **开始自动化**: 运行脚本后点击"开始"按钮
- **停止自动化**: 点击"停止"按钮或摇晃手机
- **切换模式**: 在UI中选择模式

## 图像识别配置

游戏画面因分辨率不同可能需要调整模板图片:

1. 截取游戏UI图片保存到 `res/templates/`
2. 修改 `UI.BUTTONS` 中的位置坐标

建议分辨率: 1080x1920 或更高

## 注意事项

1. **游戏更新**: 游戏更新后可能需要更新图像模板
2. **账号安全**: 过度自动化可能导致账号被封
3. **网络延迟**: 建议在WiFi环境下使用
4. **电量消耗**: 长时间运行会消耗电量

## 常见问题

### Q: 脚本无法找到游戏界面元素
A: 检查 CONFIG.logLevel 设为 'debug'，查看日志输出确认问题

### Q: 答题功能不准确
A: 诗词库可能不完整，可以到 [诗词库](#诗词库) 添加更多诗词

### Q: 如何添加新的打怪地图?
A: 编辑 MapModule.BOSS_MAPS 添加地图ID
